Motivation (II)

  • The Rule Markup Initiative has taken initial steps towards defining a shared Rule Markup Language (RuleML) for interoperation between Participants

  • RuleML permits forward (bottom-up) and backward (top-down) rules in XML for

    • deduction
    • rewriting
    • further inferential-transformational tasks

  • Current version uses backward notation, which can be applied in both directions
