Rule Markup Initiative

  • An international initiative to standardize inference rules (forward + backward) on the basis of XML

  • The need for a rule language in XML has been felt by researchers and practitioners world-wide

  • Initiative originated from discussions at PRICAI 2000, Melbourne, Australia, Aug./Sept. 2000

  • Based on proposed XML rule languages

    • BRML (Business Rules Markup Language, IBM),
    • RFML (Relational-Functional Markup Language, DFKI),
    • XRML (Extensible Rule Markup Language, KAIST)

  • Contact: Harold Boley (boley@dfki.de)
