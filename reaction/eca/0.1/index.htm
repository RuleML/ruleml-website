Overview

ECA-RuleML is an ECA rule markup language developed as part of the Rule Based Service Level Agreement Language (RBSLA) which extends RuleML with a serialization XML mak-up syntax for reactive rules and event / action constructs. It follows the design principle of RuleML and defines new constructs in separated modules which are added to RuleML on top of the hornlog layer as additional layers, i.e. ECA-RuleML adds additional expressiveness and modeling power to RuleML for the XML-based serialization of reactive rules (ECA rules), action language constructs (updates), procedural calls, event/action logics (Event Calculus) and event algebras (complex event/action pattern definitions).
