RuleML hornlog (Horn Logic) - Notes

RuleML uses Datalog as part of its family of sublanguages. A more expressive sublanguage is RuleML hornlog (Horn Logic), which includes functions with the elements <Expr> and <Fun>.

The XML syntax of RuleML 1.0 is defined by XML schemas and Relax NG schemas. Its presentation POSL can be translated online to RuleML/XML. The semantics of Datalog RuleML and Hornlog RuleML employ Herbrand models. Various Datalog and Hornlog RuleML implementations exist, including the (Naf Hornlog) RuleML reference implementation OO jDREW.
